# Astra UI Knowledge Base Navigation

This file is an index map of all detailed spec sub-files in the Astra UI design system. Jump to the corresponding document based on your specific task.

## Spec Index

| Scenario | File | Contents |
|---------|---------|---------|
| **Base Tokens** | [tokens.md](tokens.md) | Spacing, color, radius, font size, elevation, and other core variable definitions |

| **Automated Validation** | [page-agent.md](page-agent.md) | Automated test scripts, MCP integration, and rule validation |

## How to Use
When performing specific coding or design validation tasks, start by reading [validation.md](validation.md) to understand the hard rules, then view [layout.md](layout.md) based on your page type for structural guidance.
